# 🎙️ Zenith AI Voice-Controlled Assistant

A voice-controlled AI assistant that enables real-time interaction using speech input and AI-generated responses. Built using modern web technologies and integrated with the Gemini API.

---

## 🚀 Features

* 🎤 Voice input using speech-to-text
* 🧠 AI-powered responses using Gemini API
* 🔊 Text-to-speech output for natural interaction
* ⚡ Real-time processing and response generation
* ☁️ Media handling with Multer and Cloudinary

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **AI Integration:** Google Gemini API
* **Speech Processing:** Web Speech API
* **File Handling:** Multer + Cloudinary
* **Others:** REST APIs, JSON

---

## 📂 Project Structure

```
/client        → React frontend  
/server        → Node/Express backend  
/routes        → API routes  
/controllers   → Business logic  
/utils         → Helper functions  
```

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```
npm install
cd client && npm install
```

### 3. Setup environment variables

Create a `.env` file in the root:

```
GEMINI_API_KEY=your_api_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret
```

### 4. Run the application

```
npm run dev
```

---

## 🧠 How It Works

1. User gives voice input 🎤
2. Speech is converted to text
3. Text is sent to Gemini API
4. AI generates a response
5. Response is converted to speech 🔊
